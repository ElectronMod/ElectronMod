# ElectronMod

ElectronMod 是一个居于 Electron 开源项目进行二次开发的定制化应用，目的是打造一个安全高效，开箱即用的通用应用框架


## 功能特性

- 应用文件体积优化
- 应用运行性能优化
- 屏蔽相关调试入口
- 移除输入法遮罩层
- 修改应用包后缀
- 加密应用包内容
- 优化应用初始化过程
- 添加自定义接口
- 添加更多音视频编码支持
- 以管理员身份运行
- 支持应用自动更新
